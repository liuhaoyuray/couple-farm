# 我们俩的小田地 0.2.0 发布素材

## 开发版本

- 版本号：`0.2.0`
- 版本说明：`新增相恋天数、纪念日、日历提醒、体重趋势、如厕统计、资料与农场自定义及隐私数据管理。`

## 体验版重点检查

1. 两台不同微信设备首次打开后，各自设置昵称。
2. 一方生成 8 位配对码，另一方输入并绑定。
3. 双方分别记录体重和如厕，确认数据互相可见。
4. 修改昵称、头像、代表色、农场名称和相恋日期。
5. 新建、编辑、删除纪念日，并测试“加到日历”。
6. 开启称重/如厕提醒并写入系统日历；首次使用时允许日历权限。
7. 检查 7/30/90 天体重趋势、七日如厕统计和记录编辑删除。
8. 仅用测试数据验证清空本人记录、解除绑定和注销身份。

## 微信后台隐私声明参考

为实现服务，小程序可能处理以下用户主动提供或由平台生成的信息：

- 微信平台身份标识：用于自动创建并识别当前农场身份。
- 昵称、头像选择和代表色：用于双人农场内展示。
- 体重与如厕时间：用于本人记录、双方共享和趋势统计。
- 相恋日期、纪念日及备注：用于计算天数和纪念日展示。
- 提醒时间：用于应用内提示；写入手机日历必须由用户主动点击并授权。
- 配对关系与互动记录：用于向已绑定伴侣展示共享内容。

共享生活记录只向当前用户和已绑定伴侣展示；个人提醒设置不向伴侣展示。用户可以在“我们 → 隐私与数据”中清空本人记录、解除绑定或注销当前微信农场身份。

## 体验版上传后的人工步骤

自动流水线只能上传“开发版本”。管理员仍需在微信公众平台的版本管理页面将 `0.2.0` 选为体验版，并确认体验成员名单；这是微信后台账号管理操作，不能由普通代码上传密钥代替。
