# 情侣小农场 0.3.0 发布素材

## 开发版本

- 版本号：`0.3.0`
- 版本说明：`新增情侣社区、动态与趣味卡、关注点赞留言、举报屏蔽、自定义图片头像，并修复高次数趋势图。`

## 本版新增

- 情侣农场村：社区名片、全村动态和关注动态。
- 文字与单图发帖、四类话题、每日情侣问题和自愿公开的数据卡。
- 点赞、留言、关注、删除、举报、屏蔽和发布限频。
- 微信文本与图片内容安全检查；审核结果不明确时拒绝发布。
- 24 个 Emoji 头像和相册/拍照自定义头像。
- 七日如厕图改为自适应高度，高次数不会撑坏布局。
- 新用户不再使用固定昵称；社区默认关闭，趣味统计默认全部不公开。

## 体验版重点检查

1. 旧用户升级后昵称、伴侣绑定、体重、如厕和纪念日仍然存在。
2. 连续记录 8 次以上如厕，七日图仍完整显示真实次数且不溢出。
3. 在“我们”里选择图片头像并保存，伴侣端和社区动态能正常显示。
4. 进入“村口”，未开启社区时可以浏览，但不能点赞、关注、留言或发帖。
5. 开启社区并保存；确认统计默认未勾选，只有主动勾选后才进入社区卡片。
6. 分别发布纯文字、单图、每日话题和趣味数据卡动态。
7. 用另一对测试情侣验证关注、点赞、留言、举报和屏蔽。
8. 关闭社区后，原动态和留言立即从其他用户的村口消失。
9. 验证违规或审核失败内容不会公开；云调用权限首次部署后可能需要约 10 分钟生效。

## 微信后台隐私声明补充

社区功能默认关闭。用户主动开启后，可发布文字、图片、农场昵称和主动勾选的汇总数据。体重原值、如厕具体时间和次数不会自动进入社区。社区提供删除、举报、屏蔽、限频及微信内容安全检查。用户上传的头像与动态图片用于相应展示和安全审核。

## 上传后的人工步骤

自动流水线只能上传微信小程序“开发版本”。管理员仍需在微信公众平台的版本管理页面将 `0.3.0` 选为体验版，并确认体验成员；正式发布还需提交微信审核。
