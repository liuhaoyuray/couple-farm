# 我们俩的小田地 0.4.0 发布素材

## 开发版本

- 版本号：`0.4.0`
- 版本说明：`新增今天一起、餐厅抽签、每日心情和默契题、会员内测，并修复村口云端连接。`

## 本版新增

- “今天一起”成为新的底部高频入口，纪念日保留在该入口内。
- 双人每日心情与能量碰头，可留一条仅双方可见的小纸条。
- 每日默契二选一：双方选择前答案保密，两人完成后揭晓是否一致。
- 共同餐厅候选池：餐厅名、菜系、预算档位与双方共享管理。
- 三种抽签模式：完全随机、避开近 7 天、按预算筛选。
- 一次否决、双方确认与共同决定历史，解决“今晚吃什么”。
- 心动会员内测：免费层、权益层、7 天创始体验和首发优惠意向登记。
- 付费尚未接入，不会自动扣费；待微信支付商户资质完成后再启用支付。
- 修复村口集合初始化竞态、空条件查询兼容与无限量社区读取问题。
- 发布流水线新增不带用户数据的村口真实查询自检，云端未准备好时部署直接失败。

## 体验版重点检查

1. 旧用户升级后配对、记录、社区帖子和头像不丢失。
2. 进入“村口”能加载动态；空社区也能正常显示空状态。
3. 两个人分别保存今日心情，双方都能看到状态和小纸条。
4. 默契题先由一人作答时不能看到对方答案；双方作答后正确揭晓。
5. 候选池不足 2 家时不能抽签；达到 2 家后能生成结果。
6. 发起人自动同意结果，另一方确认后进入“最近吃过”。
7. 任一方否决当前结果后，可以重新抽签。
8. 免费层最多 8 家候选；领取 7 天体验后可继续添加并使用高级模式。
9. “登记首发优惠”只记录意向，不发起支付、不扣费。

## 审核说明补充

“今天一起”中的心情、小纸条、餐厅候选与默契答案仅向当前绑定的两名用户展示，不进入公开社区。社区功能仍默认关闭；只有用户主动开启后，主动发布的内容才会公开并接受微信内容安全检查。

自动流水线上传的是微信小程序“开发版本”。管理员仍需在微信公众平台把 `0.4.0` 选为体验版；正式上线前需完成隐私声明、服务类目和版本审核。
